Custom PDF Policies
Please note that encrypted PDFs will cause veraPDF execution to fail. If you are not enforcing PDF/A compliance you should check the disallow encryption policy. This will make sure that dspace uploaders will see a corresponding error message (and not just some more generic 'call system staff' message).
